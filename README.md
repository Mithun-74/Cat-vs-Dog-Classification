# 🐱🐶 Cat vs Dog Image Classification using CNN

A deep learning project that classifies images as **Cat 🐱 or Dog 🐶** using a **Convolutional Neural Network (CNN)** trained on the **Kaggle Cats vs Dogs dataset**.

The model is deployed as an interactive web application using **Gradio** and hosted on **Hugging Face Spaces**.

---

# 🚀 Live Demo

Try the application here:

👉 [https://huggingface.co/](https://huggingface.co/spaces/Mithun74/Cat_Vs_Dog_Classifier)

Upload an image of a cat or dog and the model will predict the correct class.

---

# 🧠 Project Overview

Image classification is a fundamental problem in computer vision.  
This project uses a **Convolutional Neural Network (CNN)** to classify images of cats and dogs.

The system performs the following steps:

1. Load and preprocess images
2. Resize images to **128 × 128**
3. Normalize pixel values
4. Feed the image to the trained CNN
5. Output the predicted label (**Cat or Dog**)
